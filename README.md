# DataDesk Project

Pacote de portfólio protegido do DataDesk, uma demonstração local e autocontida de engenharia de dados.

O arquivo `datadesk_project_portfolio_protegido.7z` contém o projeto completo: ingestão de documentos e XLSX, MinIO, PostgreSQL, workers OCR e PySpark, dbt, Airflow, APIs BI, Power BI, observabilidade e recovery drill.

## Acesso ao pacote

O conteúdo do arquivo está criptografado. A senha é disponibilizada apenas no currículo ou diretamente pelo autor para fins de avaliação técnica.

Para abrir no Windows, use [7-Zip](https://www.7-zip.org/). No Linux, instale o utilitário `7zip` e execute:

```bash
7zz x datadesk_project_portfolio_protegido.7z
```
